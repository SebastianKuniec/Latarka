# Latarka (Shake Flashlight)

A simple, open-source Android application that allows you to control your device's flashlight by shaking it.

---

## Features
* **Gesture Control:** Toggle the flashlight on and off by shaking the phone (Shake to toggle).
* **Background Work:** The application runs smoothly in the background via a dedicated Android Service.
* **Automated Builds:** Configured with GitHub Actions CI/CD to automatically build and generate APK files.
* **Clean Code:** No ads, no trackers, and full respect for user privacy.

---

## Requirements
* **Operating System:** Android 7.0 (API 24) or newer.
* **Hardware:** A device equipped with a camera LED flash and an accelerometer sensor.

---

## Known Issues
* **Camera ID Conflict:** Since the application continuously monitors and reads the camera ID to control the LED, opening another application that uses the camera hardware (such as the system camera app) will block access. In this scenario, the flashlight application may stop responding and will require a manual restart to function properly again.

---

## How to build & run
The project can be easily compiled using **Android Studio**:
1. Clone this repository: `git clone https://github.com/SebastianKuniec/Latarka.git`
2. Open the project in Android Studio.
3. Wait for the Gradle sync to finish, then run it on an emulator or a physical device.

Alternatively, you can download the ready-to-install `.apk` file automatically generated by GitHub Actions in the **Actions** -> **Artifacts** tab.
